## The Four Chronicles

**[I. The Wrong Hammer](./the-wrong-hammer)** — M74 · When Ollama Felt Right

The model competition began with confidence. Ollama, AMD GPU, two test questions. Within hours: Thai language bleeding into English responses at 113 seconds. A 19GB model timing out. KDE crashing at 64GB. The assumptions were clean. The results were not.

**[II. What the GTT Teaches](./what-the-gtt-teaches)** — M74 · The APU Is Not a GPU

<Term>Imperator</Term> carries a Radeon 780M — not a discrete GPU but an iGPU fused into the Ryzen 7 PRO 7840U APU. The "VRAM" is a carved slice of system RAM. The GTT is 34GB of additional system RAM the GPU can access at full memory bandwidth, with no PCIe bus in the path. Two processes fighting for that pool caused every anomaly. The Ollama logs confirmed the 19GB model had fully loaded to GPU — but the client had already given up waiting.
