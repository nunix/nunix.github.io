# ⚒ The Forge

The <Term id="forge">Forge</Term> is a standalone AIverse zone for tools that are actively being shaped. It sits beside the Idea Nebula, but it is not the same kind of place.

The **<Term id="nebula">Nebula</Term>** holds concepts before they become product-shaped.

The **<Term id="forge">Forge</Term>** holds implementations while they are still hot: prototypes, UI refinements, testing loops, performance decisions, and parallel technical tracks that may or may not become the final architecture.

DboxShim is the first artifact placed here.

<Admonition type="info" title="⚙️ Technical Insight">
The <Term id="forge">Forge</Term> is intentionally local-first. No pull request is created until the Emperor reviews the local site. This keeps the public chronicle stable while allowing the Remembrancer to write alongside implementation.

The pattern mirrors the DboxShim workflow itself: stable `main`, active feature branches, and experimental streams that only merge when the Emperor has tested them.
</Admonition>

<Admonition type="tip" title="📚 Knowledge Transfer">
**The lesson worth keeping:** document while the metal is hot, but publish only after review. The freshest technical insight appears during implementation, yet the public record must remain coherent.

**Pattern:** create a local chronicle zone for active projects; attach posts to missions; build locally; only open PR after explicit approval.
</Admonition>
