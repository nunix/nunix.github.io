# AIverse

## *In AIverse, there is only Knowledge.*

---

This is not a tutorial. This is a chronicle.

The AIverse is the story of building a **fleet of AI agents** from scratch — across multiple ships, multiple operating systems, multiple models — coordinated by a living memory called <Term id="universalis">Universalis</Term> and commanded by a hierarchy drawn from the lore of Warhammer 40,000.

It is technical storytelling. Every post is grounded in real architecture decisions, real commands, real mistakes, and the real reasons behind each choice. The WH40k skin is not decoration — it is a frame that makes the complex legible, the abstract concrete, and the mundane epic.

## The Hierarchy

The fleet is not a democracy. It is an order.

- **<Term id="emperor">The Emperor</Term>** — absolute authority. The human. nunix. His word shapes every mission.
- **<Term id="general">The General</Term>** — fleet captain. Receives, delegates, analyzes, reports. Never does field work.
- **<Term id="matey">Mateys</Term>** — worker agents (Claude Haiku) on each ship. Execute. Record. Return.

## The Memory

<Term id="universalis">Universalis</Term> is the fleet's brain — a PostgreSQL database where every mission, delegation, observation, and correction is recorded. Every entry has a `parent_id`, building a graph of causality across the fleet's entire history.

The fleet does not forget. The fleet learns.
